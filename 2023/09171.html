第一讲：基础算法

1.1.快速排序

1.1.1 快速排序的基本思想（分治）

1.选取基准值，通过不同的方式挑选出基准值。(可以是左端点、右端点、中间值，也可以是随机值)
2.用分治的思想进行分割，通过该基准值在序列中的位置，将序列分成两个区间，在基准值左边的区间里的数都比基准值小（默认以升序排序），在基准值右边的区间里的数都比基准值大。
3.递归调用快速排序的函数对两个区间再进行上两步操作，直到调用的区间为空或只有一个数。

1.1.2 快速排序的代码实现

void quick_sort(int q[], int l, int r)
{
    //递归的终止情况
    if(l >= r) return;

    //第一步：分成子问题
    int i = l - 1, j = r + 1;
    //如果区间取[l,i-1]和[i,r]这种，那么x不应该取左边界(l、(l+r)/2)。应取 x = q[r]; x = [(l+r+1)/2];
    //如果区间取[l,j]和[j+1,r]这种，那么x不应该取右边界(如r、(l+r+1)/2)。应取 x = q[l]; x = q[(l+r)/2];
    int x = q[l + r >> 1]; // 基准值
    while(i < j) 
    {
        do i++; while(q[i] < x);
        do j--; while(q[j] > x);
        if(i < j) swap(q[i], q[j]);
    }

    //第二步：递归处理子问题
    quick_sort(q, l, j), quick_sort(q, j + 1, r);

}

1.1.3 快速排序的模板题

一、

题目：785. 快速排序

给定你一个长度为 n 的整数数列。请你使用快速排序对这个数列按照从小到大进行排序。并将排好序的数列按顺序输出。

输入格式

输入共两行，第一行包含整数 n。第二行包含 n个整数（所有整数均在 1∼109范围内），表示整个数列。

输出格式

输出共一行，包含 n 个整数，表示排好序的数列。

数据范围

$ 1 ≤ N ≤ 100000 $

输入样例：

5
3 1 2 4 5

输出样例：

1 2 3 4 5

解答

import java.io.*;
import java.util.*;

public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    
    static void quick_sort(int[] a,int l,int r){
        if(l>=r) return; //递归的退出条件;
        //分成子问题
        int i=l-1,j=r+1;
        int x=a[l+r>>1];//基准值 
        while(i<j){
            do i++; while(a[i]<x); //
            do j--; while(a[j]>x);
            if(i<j){
                int temp=a[i];
                a[i]=a[j];
                a[j]=temp;
            }
        }
        //递归处理子问题
        quick_sort(a,l,j);
        quick_sort(a,j+1,r);
        
    }
    
    public static void main(String args[]) throws IOException {
        int n=Integer.parseInt(in.readLine());
        int[] a=new int[n];
        String[] nums=in.readLine().split(" ");
        for(int i=0;i<nums.length;i++){
            a[i]=Integer.parseInt(nums[i]);
        }
        quick_sort(a,0,n-1);
        for(int i=0;i<a.length;i++){
            out.print(a[i]+" ");
        }
        in.close();
        out.close();
    }
}

二、

题目：786. 第k个数

给定一个长度为 n 的整数数列，以及一个整数 k，请用快速选择算法求出数列从小到大排序后的第 k 个数。

输入格式
第一行包含两个整数 n 和 k 。
第二行包含 n 个整数（所有整数均在 1∼109 范围内），表示整数数列。

输出格式
输出一个整数，表示数列的第 k 小数。

数据范围
$1≤n≤100000$
$1≤k≤n$

输入样例：

5 3
2 4 1 5 3

输出样例：

3

解答

import java.io.*;

public class Main{
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    
    static int quick_sort(int[] a,int l,int r,int k){
        if(l>=r) return a[l]; //递归的退出条件
        int i=l-1,j=r+1;
        int x=a[l+r>>1]; //基准值
        while(i<j){
            do i++; while(a[i]<x);
            do j--; while(a[j]>x);
            if(i<j){
                int temp=a[i];
                a[i]=a[j];
                a[j]=temp;
            }
        }
        int len=j-l+1; //算出分界点j与左边界点l之间的距离
        if(k<=len) {//如果k比len小说明，第K个数在分界点的左边,就递归左边进行寻找
            return quick_sort(a,l,j,k);
        }
        //否则，说明第K个数在分界点的右边,就递归右边进行寻找
        return quick_sort(a,j+1,r,k-len);
    }
    
    public static void main(String args[]) throws IOException{
        String[] nums=in.readLine().split(" ");
        int n=Integer.parseInt(nums[0]);
        int k=Integer.parseInt(nums[1]);
        int[] a=new int[n];
        String[] arr=in.readLine().split(" ");
        for(int i=0;i<arr.length;i++){
            a[i]=Integer.parseInt(arr[i]);
        }
        
        int res=quick_sort(a,0,n-1,k);
        out.println(res);
        in.close();
        out.close();
    }
}

1.2.归并排序

1.2.1 归并排序的基本思想(分治)

归并排序算法有两个基本的操作，一个是分，也就是把原数组划分成两个子数组的过程。另一个是治，它将两个有序数组合并成一个更大的有序数组。
    1.将待排序的线性表不断地切分成若干个子表，直到每个子表只包含一个元素，这时，可以认为只包含一个元素的子表是有序表。
    2.将子表两两合并，每合并一次，就会产生一个新的且更长的有序表，重复这一步骤，直到最后只剩下一个子表，这个子表就是排好序的线性表。


1.2.2 归并排序的代码实现

void merge_sort(int q[], int l, int r)
{
    //递归的终止情况
    if(l >= r) return;

    //第一步：分成子问题
    int mid = l + r >> 1;

    //第二步：递归处理子问题
    merge_sort(q, l, mid ), merge_sort(q, mid + 1, r);

    //第三步：合并子问题
    int k = 0, i = l, j = mid + 1, tmp[r - l + 1];
    while(i <= mid && j <= r)
        if(q[i] <= q[j]) tmp[k++] = q[i++];
        else tmp[k++] = q[j++];
    while(i <= mid) tmp[k++] = q[i++];
    while(j <= r) tmp[k++] = q[j++];

    for(k = 0, i = l; i <= r; k++, i++) q[i] = tmp[k];
}

1.2.3 归并排序的模板题

一、

题目：787. 归并排序

给定你一个长度为 n的整数数列。请你使用归并排序对这个数列按照从小到大进行排序。并将排好序的数列按顺序输出。

输入格式
输入共两行，第一行包含整数 n。

第二行包含 n 个整数（所有整数均在 1∼1e9 范围内），表示整个数列。

输出格式
输出共一行，包含 n 个整数，表示排好序的数列。

数据范围
$1≤n≤100000$

输入样例：

5
3 1 2 4 5

输出样例：

1 2 3 4 5

解决

import java.io.*;
public class Main{
    static BufferedReader in =new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=100010;
    static int[] arr=new int[N];
    static int[] temp=new int[N];
    
    static void merge_sort(int[] arr,int l,int r){
        if(l>=r) return;//递归的退出条件
        //分成子问题
        int mid=l+r>>1;
        //递归处理子问题 
        merge_sort(arr,l,mid);
        merge_sort(arr,mid+1,r);
        //合并子问题
        int i=l,j=mid+1;
        int k=0;
        //比较两个有序序列
        while(i<=mid&&j<=r){
            if(arr[i]<=arr[j]){
                temp[k++]=arr[i++];
            }else{
                temp[k++]=arr[j++];
            }
        }
        //把剩下的一个序列的数据统统拷贝到temp数组中
        while(i<=mid){
            temp[k++]=arr[i++];
        }
        while(j<=r){
            temp[k++]=arr[j++];
        }
        //把临时数组temp拷贝回去
        for(i=l,k=0;i<=r;i++,k++){
            arr[i]=temp[k];
        }
        
    }
        
    
    public static void main(String args[]) throws IOException{
        int n=Integer.parseInt(in.readLine());
        String[] nums=in.readLine().split(" ");
        for(int i=0;i<nums.length;i++){
            arr[i]=Integer.parseInt(nums[i]);
        }
        merge_sort(arr,0,n-1);
        for(int i=0;i<n;i++){
            out.print(arr[i]+" ");
        }
        in.close();
        out.close();
    }
}

二、

题目：788. 逆序对的数量

给定一个长度为 n 的整数数列，请你计算数列中的逆序对的数量。

逆序对的定义如下：对于数列的第 i 个和第 j 个元素，如果满足 $i<j$ 且 $a[i]>a[j]$
，则其为一个逆序对；否则不是。

输入格式
第一行包含整数 n ，表示数列的长度。
第二行包含 n 个整数，表示整个数列。

输出格式
输出一个整数，表示逆序对的个数。

数据范围
$1≤n≤100000$，
数列中的元素的取值范围 $[1,109]$。

输入样例：

6
2 3 4 5 6 1

输出样例：

5

解答

import java.io.*;
public class Main{
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=100000;
    static int[] arr=new int[N];
    static int[] temp=new int[N];
    
    static long merge_sort(int[] arr,int l,int r) {
        if(l>=r) return 0; //递归的退出条件
        //划分子问题
        int mid=l+r>>1;
        long res=merge_sort(arr,l,mid)+merge_sort(arr,mid+1,r);
        //递归处理子问题
        int i=l,j=mid+1;
        int k=0;
        // 合并两个有序序列 
        while(i<=mid&&j<=r){ 
            if(arr[i]<=arr[j]){
                temp[k++]=arr[i++];
            }else {
                temp[k++]=arr[j++];
                res +=mid-i+1;//若此时左区间指向的数大于右区间指向的数，故逆序对的个数有mid-i+1个数.
            }
        }
        //把剩下的有序序列都拷贝到temp中
        while(i<=mid){
            temp[k++]=arr[i++];
        }
        while(j<=r) {
            temp[k++]=arr[j++];
        }
        //把临时数组都拷贝回去
        for(i=l,k=0;i<=r;i++,k++){
            arr[i]=temp[k];
        }
        return res;
    }
    
    public static void main(String args[]) throws IOException{
        int n = Integer.parseInt(in.readLine());
        String[] nums=in.readLine().split(" ");
        for(int i=0;i<nums.length;i++){
            arr[i]=Integer.parseInt(nums[i]);
        }
        long res = merge_sort(arr,0,n-1);
        out.println(res);
        in.close();
        out.close();
    }
}

1.3.二分

1.3.1 二分的基本思想

在一个区间上，找到某种性质，每次可以将区间一分为二（存在边界），一个区间满足、另外一个区间不满足，答案就在边界上！(单调一定可以二分，不单调的有的也可以二分)

1.3.2 二分的代码实现

// 1.整数二分算法模板
bool check(int x) {/* ... */} // 检查x是否满足某种性质
// 区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int SL(int l, int r)//(1)查找右部分的左边界 SearchLeft 简写SL
{
    while (l < r)
    {
        int mid = l + r >> 1;
        if (check(mid)) r = mid; // check()判断mid是否满足性质
        else l = mid + 1;//左加右减
    }
    return l;
}
// 区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int SR(int l, int r)//(2)查找左部分的右边界 SearchRight 简写SR 
{
    while (l < r)
    {
        int mid = l + r + 1 >> 1;//如果下方else后面是l则这里加1
        if (check(mid)) l = mid;
        else r = mid - 1;//左加右减
    }
    return l;
}

//2.浮点数二分算法模板
bool check(double x) {/* ... */} // 检查x是否满足某种性质

double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}

一、

题目：789. 数的范围

给定一个按照升序排列的长度为 $n$ 的整数数组，以及 $q$ 个查询。
对于每个查询，返回一个元素 $k$ 的起始位置和终止位置（位置从 $0$ 开始计数）。
如果数组中不存在该元素，则返回 -1 -1。

输入格式
第一行包含整数 $n$ 和 $q$ ，表示数组长度和询问个数。

第二行包含 $n$ 个整数（均在 $1∼10000$ 范围内），表示完整数组。

接下来 $q$ 行，每行包含一个整数 $k$，表示一个询问元素。

输出格式
共 $q$ 行，每行包含两个整数，表示所求元素的起始位置和终止位置。

如果数组中不存在该元素，则返回 -1 -1。

数据范围
$1≤n≤100000$
$1≤q≤10000$
$1≤k≤10000$

输入样例:

6 3
1 2 2 3 3 4
3
4
5

输出样例：

3 4
5 5
-1 -1

解答

import java.io.*;
public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out=new PrintWriter(new OutputStreamWriter(System.out));
    static int n,q;
    static final int N=100010;
    static int[] arr=new int[N];
    public static void main(String args[]) throws IOException{
        String[] nums= in.readLine().split(" ");
        n=Integer.parseInt(nums[0]);
        q=Integer.parseInt(nums[1]);
        String[] info= in.readLine().split(" ");
        for(int i=0;i<info.length;i++){
            arr[i]=Integer.parseInt(info[i]);
        }
        while((q--)>0) {
            int x=Integer.parseInt(in.readLine());
            int l=0,r=n-1;
            while(l<r) {
                int mid=l+r>>1;
                if(arr[mid]>=x){//左边界限制
                    r=mid;
                }else{
                    l=mid+1;
                }
            }
            if(arr[l]==x){
                out.print(l+" ");
                r=n-1;
                while(l<r){
                    int mid=l+r+1>>1;
                    if(arr[mid]<=x){//右边界限制
                        l=mid;
                    }else{
                        r=mid-1;
                    }
                }
                out.println(r);
            }else{
                out.println("-1 -1");
            }
        }
        in.close();
        out.close();
    }
}

题目：790. 数的三次方根

给定一个浮点数 $n$ ，求它的三次方根。

输入格式
共一行，包含一个浮点数 $n$ 。

输出格式
共一行，包含一个浮点数，表示问题的解。
注意，结果保留 $6$ 位小数。

数据范围
$−10000≤n≤10000$

输入样例：
1000.00

输出样例：
10.000000

解答

import java.io.*;
public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out=new PrintWriter(new OutputStreamWriter(System.out));
    
    public static void main(String args[]) throws IOException{
         double n=Double.parseDouble(in.readLine());
         double l=-10000,r=10000;
         while(r-l>1e-8){
             double mid=(l+r)/2;
             if(mid*mid*mid>=n){
                 r=mid;
             }else{
                 l=mid;
             }
         }
         out.println(String.format("%.6f",l));
         in.close();
         out.close();
    }
}

三、

题目：730. 机器人跳跃问题

机器人正在玩一个古老的基于 DOS 的游戏。

游戏中有 $N+1$ 座建筑——从 $0$ 到 $N$ 编号，从左到右排列。

编号为 $0$ 的建筑高度为 $0$ 个单位，编号为 $i$ 的建筑高度为 $H(i)$ 个单位。

起初，机器人在编号为 $0$ 的建筑处。

每一步，它跳到下一个（右边）建筑。

假设机器人在第 k 个建筑，且它现在的能量值是 E ，下一步它将跳到第 $k+1$ 个建筑。

如果 $H(k+1)>E$ ，那么机器人就失去 $H(k+1)−E$ 的能量值，否则它将得到 $E−H(k+1)$ 的能量值。

游戏目标是到达第 $N$ 个建筑，在这个过程中能量值不能为负数个单位。

现在的问题是机器人至少以多少能量值开始游戏，才可以保证成功完成游戏？

输入格式
第一行输入整数 $N$。

第二行是 $N$ 个空格分隔的整数，$H(1),H(2),…,H(N)$ 代表建筑物的高度。

输出格式
输出一个整数，表示所需的最少单位的初始能量值上取整后的结果。

数据范围
$1≤N,H(i)≤105$

输入样例1：

5
3 4 3 2 4

输出样例1：

4

输入样例2：

3
4 4 4

输出样例2：

4

输入样例3：

3
1 6 4

输出样例3：

3

解答

import java.io.*;
public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out=new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=100010;
    static int n;
    static int[] arr=new int[N];
    
    static boolean check(int e) { 
        for(int i=0;i<n;i++){
            e=e*2-arr[i];
            if(e>=1e5) return true;//如果E大于最大值，后面的数值必然大于0
            if(e<0) return false;
        }
        return true;
    }
    
    public static void main(String args[]) throws IOException{
        n=Integer.parseInt(in.readLine());
        String[] nums=in.readLine().split(" ");
        for(int i=0;i<nums.length;i++){
            arr[i]=Integer.parseInt(nums[i]);
        }
        int l=0,r=100000;//从0到100000之间寻找E（0是E的最小值，100000是E的最大值）
        while(l<r){
            int mid=l+r>>1;
            if(check(mid)){
                r=mid;
            }else{
                l=mid+1;
            }            
        }
        out.println(r);
        in.close();
        out.close();
    }
}

题目: 1221. 四平方和

四平方和定理，又称为拉格朗日定理：

每个正整数都可以表示为至多 $4$ 个正整数的平方和。

如果把 $0$ 包括进去，就正好可以表示 $4$ 个数的平方和。

比如：

$ 5=0^2+0^2+1^2+2^2 $
$ 7=1^2+1^2+1^2+2^2 $

对于一个给定的正整数，可能存在多种平方和的表示法。

要求你对 $4$ 个数排序：
$0≤a≤b≤c≤d$
并对所有的可能表示法按 $a,b,c,d$为联合主键升序排列，最后输出第一个表示法。

输入格式
输入一个正整数 $N$。

输出格式
输出$4$个非负整数，按从小到大排序，中间用空格分开。

数据范围
$0<N<5∗10^6$

输入样例：

5

输出样例：

0 0 1 2

解答

//二分
import java.io.*;
import java.util.*;
public class Main{
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    public static void main(String args[]) throws IOException{
        int n=Integer.parseInt(in.readLine());
        Sum[] sum= new Sum[5000000];
        int i=0;
        //枚举C和D这两个数
        for(int c=0;c*c<=n;c++){
            for(int d=c;c*c+d*d<=n;d++){
                sum[i++]=new Sum(c*c+d*d,c,d);
            }
        }
        //按字典序排序
        Arrays.sort(sum,0,i);
        //枚举A和B这两个数
        for(int a=0;a*a<=n;a++){
            for(int b=a;a*a+b*b<=n;b++){
                int t=n-a*a-b*b;
                //通过t和sum数组里面的s元素进行对比,因为sum的s元素是单调的，因此可以考虑二分查找
                int l=0,r=i-1;
                while(l<r){
                    int mid=l+r>>1;
                    if(sum[mid].s>=t){
                        r=mid;
                    }else{
                        l=mid+1;
                    }
                }
                if(sum[l].s==t){
                    System.out.printf("%d %d %d %d",a,b,sum[l].c,sum[l].d);
                    return;
                }
            }
        }
    }
}
class Sum implements Comparable<Sum> {
    int s,c,d;
    public Sum(int s,int c,int d) {
        this.s=s;
        this.c=c;
        this.d=d;
    }
    @Override
    public int compareTo(Sum o) {
        if(s!=o.s) return Integer.compare(s,o.s);
        if(c!=o.c) return Integer.compare(c,o.c);
        return Integer.compare(d,o.d);
    }
}

        for(int c=0;c*c<=n;c++){
            for(int d=c;c*c+d*d<=n;d++){
                int t=c*c+d*d;
                if(map.get(t)==null){//保证了按字典序排序
                    map.put(t,new int[]{c,d});
                }
            }
        }
        //枚举a和b两个数
        for(int a=0;a*a<=n;a++){
            for(int b=a;a*a+b*b<=n;b++){
                int res=n-a*a-b*b;
                if(map.get(res)!=null){
                    int c=map.get(res)[0];
                    int d=map.get(res)[1];
                    System.out.printf("%d %d %d %d",a,b,c,d);
                    return;
                }
            }
        }
    }
}

五、

题目：1227. 分巧克力

儿童节那天有 $K$ 位小朋友到小明家做客。

小明拿出了珍藏的巧克力招待小朋友们。

小明一共有 $N$ 块巧克力，其中第 $i$ 块是 $Hi×Wi$ 的方格组成的长方形。

为了公平起见，小明需要从这 $N$ 块巧克力中切出 $K$ 块巧克力分给小朋友们。

切出的巧克力需要满足：

1.形状是正方形，边长是整数
2.大小相同

例如一块 $6×5$ 的巧克力可以切出 $6$ 块 $2×2$的巧克力或者 $2$ 块 $3×3$ 的巧克力。

当然小朋友们都希望得到的巧克力尽可能大，你能帮小明计算出最大的边长是多少么？

输入格式

第一行包含两个整数 $N$ 和 $K$。

以下 $N$ 行每行包含两个整数 $Hi$ 和 $Wi$。

输入保证每位小朋友至少能获得一块 $1×1$ 的巧克力。

输出格式

输出切出的正方形巧克力最大可能的边长。

数据范围

$
1≤N,K≤105,
$
$
1≤Hi,Wi≤105
$

输入样例：

2 10
6 5
5 6

输出样例：

2

解答 

import java.io.*;
public class Main{
    static int n,k;
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=100010;
    static int[] h=new int[N],w=new int[N];
    
    static boolean check(int mid) {
        int res=0;
        for(int i=0;i<n;i++){
            res +=(h[i]/mid)*(w[i]/mid);
            if(res>=k) return true;
        }
        return false;
    }
    
    public static void main(String args[]) throws IOException{
        String[] nums=in.readLine().split(" ");
        n=Integer.parseInt(nums[0]);
        k=Integer.parseInt(nums[1]);
        
        for(int i=0;i<n;i++){
            String[] info=in.readLine().split(" ");
            h[i]=Integer.parseInt(info[0]);
            w[i]=Integer.parseInt(info[1]);
        }
        
        int l=1,r=100000;//二分的初始范围
        while(l<r){
            int mid=l+r+1>>1;//巧克力的边长
            if(check(mid)){//满足的话，说明当前的边长有可能不是最大的，是要继续往右边找
                l=mid;
            }else{//否则就往左边找
                r=mid-1;
            }
        }
        out.println(r);
        in.close();
        out.close();
    }
}

1.4.前缀和

1.3.1 前缀和的基本思想

原数组: $a[1], a[2], a[3], a[4], a[5], …, a[n]$
前缀和 $Si$ 为数组的前 $i$ 项和
前缀和: $S[i] = a[1] + a[2] + a[3] + … + a[i]$

注意: 前缀和的下标最好从 1 开始, 避免进行下标的转换

s[0] = 0
s[1] = a[1]
s[2] = a[1] + a[2]

前缀和的作用
快速求出元素组中某段区间的和

一维数组求解前缀和($Si$)
$for$ 循环求出每个$S[i]$ (将 $S[0]$ 定义为 $0$, 避免下标的转换)
求 [l, r]中的和, 即为 $S[r] - S[l-1]$


二维数组求解前缀项和

1.求解 $s[i][j]$
$s[i][j]=s[i-1][j]+s[i][j-1]-s[i-1][j-1]+a[i][j]$


2.求解 $s[i][j]$ 的子矩阵
子矩阵和为：$s[x2][y2]-s[x2][y1-1]-s[x1-1][y2]+s[x1-1][y1-1]$

1.3.2 前缀和的几道经典题

一、

题目: 795. 前缀和

输入一个长度为 $n$ 的整数序列。

接下来再输入 $m$ 个询问，每个询问输入一对 $l,r$ 。

对于每个询问，输出原序列中从第 $l$ 个数到第 $r$ 个数的和。

输入格式

第一行包含两个整数 $n$ 和 $m$。

第二行包含 $n$ 个整数，表示整数数列。

接下来 $m$ 行，每行包含两个整数 $l$ 和 $r$ ，表示一个询问的区间范围。

输出格式
共 $m$ 行，每行输出一个询问的结果。

数据范围
$
1≤l≤r≤n,
$
$
1≤n,m≤100000,
$
$
−1000≤数列中元素的值≤1000
$

输入样例：

5 3
2 1 3 6 4
1 2
1 3
2 4

输出样例：

3
6
10

解答

import java.io.*;

public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out=new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=100010;
    static int[] a=new int[N],s=new int[N];
    
    public static void main(String args[]) throws IOException{
        String[] nums=in.readLine().split(" ");
        int n=Integer.parseInt(nums[0]);
        int m=Integer.parseInt(nums[1]);
        String[] info=in.readLine().split(" ");
        for(int i=0;i<n;i++){
            a[i+1]=Integer.parseInt(info[i]);
        }
        for(int i=1;i<=n;i++){
            s[i]=s[i-1]+a[i];//算出各项的前缀和
        }
        
        while((m--)>0){
            String[] s1=in.readLine().split(" ");
            int l=Integer.parseInt(s1[0]);
            int r=Integer.parseInt(s1[1]);
            out.println(s[r]-s[l-1]);//再求某一个序列的区间和
        }
        in.close();
        out.close();
    }
}

二、

题目：796. 子矩阵的和

输入一个 $n$ 行 $m$ 列的整数矩阵，再输入 $q$ 个询问，每个询问包含四个整数 $x1,y1,x2,y2$ ，表示一个子矩阵的左上角坐标和右下角坐标。

对于每个询问输出子矩阵中所有数的和。

输入格式
第一行包含三个整数 $n，m，q$。

接下来 $n$ 行，每行包含 $m$ 个整数，表示整数矩阵。

接下来 $q$行，每行包含四个整数 $x1,y1,x2,y2$ ，表示一组询问。

输出格式
共 $q$ 行，每行输出一个询问的结果。

数据范围
$1≤n,m≤1000,$
$1≤q≤200000,$
$1≤x1≤x2≤n,$
$1≤y1≤y2≤m,$
$−1000≤矩阵内元素的值≤1000$

输入样例：

3 4 3
1 7 2 4
3 6 2 8
2 1 2 3
1 1 2 2
2 1 3 4
1 3 3 4

输出样例：

17
27
21

解答

import java.io.*;
public class Main {
    static BufferedReader in=new BufferedReader(new InputStreamReader(System.in));
    static PrintWriter out =new PrintWriter(new OutputStreamWriter(System.out));
    static final int N=1010;
    static int[][] a=new int[N][N],s=new int[N][N];
    
    public static void main(String args[]) throws IOException{
        String[] nums=in.readLine().split(" ");
        int n=Integer.parseInt(nums[0]);
        int m=Integer.parseInt(nums[1]);
        int q=Integer.parseInt(nums[2]);
        for(int i=0;i<n;i++){
            String[] info= in.readLine().split(" ");
            for(int j=0;j<m;j++){
                a[i+1][j+1]=Integer.parseInt(info[j]);
            }
        }
        for(int i=1;i<=n;i++){
            for(int j=1;j<=m;j++){
                s[i][j]=s[i-1][j]+s[i][j-1]-s[i-1][j-1]+a[i][j];//求二维前缀和
            }
        }
        
        while((q--)>0) {
            String[] info1=in.readLine().split(" ");
            int x1=Integer.parseInt(info1[0]);
            int y1=Integer.parseInt(info1[1]);
            int x2=Integer.parseInt(info1[2]);
            int y2=Integer.parseInt(info1[3]);
            out.println(s[x2][y2]-s[x1-1][y2]-s[x2][y1-1]+s[x1-1][y1-1]);//求子矩阵的二维前缀和
        }
        in.close();
        out.close();
    }
}
